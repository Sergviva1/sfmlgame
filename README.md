Space Adventure Game

Простая 2D игра на C++ с использованием библиотеки SFML, в которой игрок управляет космическим кораблем, избегает метеоритов, собирает бонусы и монеты.

Для запуска запустить main.exe

Управление
В главном меню:

    ENTER - Начать игру

    ESC - Выйти из игры

Во время игры:

    WASD - Движение корабля

    N - Смена музыкального трека

    ESC - Выход из игры
